<h1 align="center">💲 Conversor de arquivos .csv para PDF e HTML 💲</h1>

<p align="center">Esse código transforma dados salvos em .csv em PDF e HTML.</p>


<hr>
<h2  align="center">🎁 Tecnologias utilizadas 🎁</h2>
<p align="center"> 
Node.js <br>
EJS <br></p>

<hr>
<h2  align="center">🎁 Como usar? 🎁</h2>
<p align="center"> 
Execute <code>npm install</code> no terminal na pasta do projeto.<br/>
Em seguida execute <code>node index.js</code> e veja o resultado na pasta output. <br/>
O arquivo .csv que ele irá ler estará na pasta input. 
</p>
